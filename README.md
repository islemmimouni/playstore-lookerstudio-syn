
# Play Store → Looker Studio Sync

## 📌 Description
Ce projet permet de synchroniser automatiquement les rapports de la Google Play Console vers Looker Studio pour créer des tableaux de bord analytiques.

## 🛠️ Technologies utilisées
- Google Cloud Storage API
- Google BigQuery
- Looker Studio

## 📋 Prérequis
- Compte développeur Google Play
- Accès à Google Cloud Platform
- Bucket ID de la Play Console
